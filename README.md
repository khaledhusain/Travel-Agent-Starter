# AI Travel Planning Agent

A simple Python-based travel planning agent that generates realistic, budget-aware, multi-day travel itineraries using the OpenAI API.

This project is a demo focused on **API integration**, **prompt-driven agents**, and clean backend logic rather than UI.


## Features

- Generates day-by-day travel itineraries based on origin, destination, duration, and budget
- Keeps recommendations realistic and budget-conscious
- Uses environment variables for secure API key management


## Tech Stack

- Python
- OpenAI API
- `python-dotenv`
